The Mediterranean Pack contains Four animals and Four Plant from the Mediterranean Region, both Aquatic and Terrestrial.

# Requirements

Zt2 + ES and MM2 requiered. (EA for Black pine, but not esential)

# Includes

## Animals

- Alpine Marmot
- Spanish Ibex
- Iberian Wolf
- Dalmatian Pelican

## Foliage

- European Black Pine
- Cork Oak
- Spanish Lavender
- Allepo Pine.

# Installation

To Install place the z2f file in `c:\Program Files\Microsoft Games\Zoo Tycoon 2`

# Known Issues

- On my test I had trouble locating the Alpine Marmot in the zoopedia or the adopt menu. Could be an xml coding error. Otherwise, the rest of the pack seems to work as described.
